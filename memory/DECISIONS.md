# 决策记录

## 2026-03-26

- 项目名称沿用 `TrailNote`
- 设计与交互基线以 `prototype/` 原型与 `tokens.json` 为准
- 客户端不使用 uni-app，采用各端原生技术栈
- iOS 使用 Swift / SwiftUI 方向
- Android 使用 Kotlin / Jetpack Compose 方向
- H5 使用原生 Web 技术栈，建议 `Vue 3 + TypeScript`
- 微信小程序使用原生小程序技术栈
- 服务端主数据库选型为 MySQL
- 移动端本地离线数据库定位为 SQLite
- 后台管理前端基于现有 `vue-vben-admin` 脚手架继续开发
- 服务端工程管理工具先沿用 Maven 多模块结构
- H5 正式工程继续承接 `prototype/` 的设计 token 与交互规则，作为多端统一 UI 的首个可运行基线
